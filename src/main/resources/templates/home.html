Fast

Reactor's Selector-based dispatching mechanism can pump upwards of 15,000,000 events per second into your components using the LMAX Disruptor-based RingBufferDispatcher.

Scalable

Reactor's components are designed to be minimalist and consume very few resources to maximize CPU and memory usage in constrained environments like on cloud servers or in VMs.

Simple

Reactor has a simple and straightforward API for wiring event handlers which Stream and Promise build on to provide functional-style composition for succinct and readable code.
